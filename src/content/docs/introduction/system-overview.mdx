## UI 레이아웃

GAEMI Loci의 화면은 **사이드바**, **상단 탭**, **메인 편집 영역**, **하단 상태 바**로 구성됩니다.

### 상단 좌측 탭

<Image src={screenshotTabLeft} alt="상단 좌측 탭 — Overview와 Workspace 두 개의 탭으로 구성" />

대시보드에서 **Overview**와 **Workspace** 두 가지 보기 모드를 전환합니다.

- **Overview**: 전체 Job 현황을 Phase별로 한눈에 확인하고, 세계 지도 위에서 Job 위치를 파악합니다.
- **Workspace**: 선택한 Job의 상세 워크플로를 관리하고, 각 Step을 진행합니다.

### 상단 우측 탭

<Image src={screenshotTabRight} alt="상단 우측 탭 — Phase별 Job 수, HD Maps/Nav Maps 수, CPU/RAM 사용률 실시간 표시" />

시스템 상태를 실시간으로 모니터링합니다.

| 항목 | 설명 |
|------|------|
| **P1 / P2 / P3** | 각 Phase에 있는 Job 수 (색상으로 Phase 구분) |
| **HD Maps / Nav Maps** | 생성된 맵 수 |
| **CPU / RAM** | 서버 리소스 사용률 (게이지 바로 시각화) |

## 제공 도구

GAEMI Loci는 HD 맵 생성의 각 단계에 맞는 전용 편집 도구를 제공합니다.

<img src={editorsOverview.src} alt="GAEMI Loci 에디터 모듈 개요" />

<CardGrid>
  <Card title="대시보드" icon="laptop">
    맵 생성 작업(Job)을 만들고, 각 단계의 진행 상태를 한눈에 확인합니다.
  </Card>
  <Card title="서비스 구역 에디터" icon="pencil">
    위성 지도 위에서 로봇이 운행할 서비스 구역을 지정하고 등록합니다. 주소 검색과 위치 선택을 지원합니다.
  </Card>
  <Card title="내비게이션 에디터" icon="random">
    로봇이 따라갈 주행 경로를 편집합니다. 경유지(노드)와 구간(엣지)을 추가하거나 수정할 수 있습니다.
  </Card>
  <Card title="베이스 맵 에디터" icon="open-book">
    현장에서 수집한 3D 데이터를 확인하고, 충전 스테이션과 기준점을 등록합니다.
  </Card>
  <Card title="3D 맵 에디터 / 맵 보정" icon="setting">
    기준점 정렬, 경로 보정, 맵 병합, 장애물 맵 편집 등의 도구로 맵 정확도를 개선합니다.
  </Card>
  <Card title="파일 관리" icon="seti:folder">
    매핑 데이터를 업로드하거나, 완성된 맵 파일을 다운로드합니다.
  </Card>
</CardGrid>

## 사용자 역할

GAEMI Loci는 역할에 따라 사용하는 도구가 다릅니다.

| 역할 | 담당 업무 | 주요 사용 도구 |
|------|----------|--------------|
| <Badge text="Biz & Ops" variant="note" /> | 서비스 구역 등록, 워크플로 관리 | 대시보드, 서비스 구역 에디터 |
| <Badge text="Path Editor" variant="success" /> | 로봇 주행 경로 작성 및 편집 | 내비게이션 에디터 |
| <Badge text="Map Editor" variant="caution" /> | 맵 편집 및 품질 보정 | 베이스 맵 에디터, 3D 맵 에디터 / 맵 보정 |
| <Badge text="Inspector" variant="danger" /> | 맵 품질 검수 및 승인 | 베이스 맵 에디터 |

<Aside>
하나의 계정이 여러 역할을 겸할 수 있습니다. 예를 들어, 소규모 현장에서는 한 사람이 경로 편집과 맵 편집을 모두 담당할 수 있습니다.
</Aside>
