## Key Features

<CardGrid>
  <Card title="Step-by-Step Map Creation Guide" icon="laptop">
    Guides you through every stage of the workflow — from defining service areas to HD map generation, map editing, and robot deployment.
  </Card>
  <Card title="Intuitive Map Editing Tools" icon="pencil">
    Offers purpose-built editors such as satellite maps, 3D views, and 2D editors, making it easy for anyone to refine a map.
  </Card>
  <Card title="Map Quality Refinement" icon="setting">
    Improve map accuracy with features such as ground control point alignment, path correction, and map merging.
  </Card>
  <Card title="Cloud-Based Automated Processing" icon="random">
    Upload the data collected in the field, and the server will automatically generate an HD map. You can track progress in real time.
  </Card>
</CardGrid>
