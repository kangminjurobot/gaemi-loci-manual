The dashboard is the central hub for managing the HD Map generation workflow. You can create Jobs, track their progress, and launch tools all from one place.

## Key Concepts

| Concept | Description |
|------|------|
| **Phase** | The top-level stage of the workflow. The workflow consists of three Phases. |
| **Job** | A unit of HD Map generation work for a single service area. Each Job follows its own independent workflow. |
| **Step** | An individual task within a workflow. Each Phase contains several Steps. |
| **Session** | The unit for each iteration when expanding the map repeatedly in Phase 3. |

The three Phases are organized as follows:

- **Phase 1 — Pre-installation**: Define the service area, then create and review the robot path.
- **Phase 2 — On-site Installation**: Collect mapping data in the field, run data processing and map generation, and perform map editing.
- **Phase 3 — Area Expansion**: Add new areas to an existing map. Expand the map incrementally by repeating the process session by session.
