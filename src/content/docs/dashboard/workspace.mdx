Workspace 탭은 선택된 Job의 상세 워크플로를 관리하는 작업 공간입니다. 좌측에 Job 목록이, 우측에 Phase별 Step 카드가 표시됩니다.

## Phase 1 — Setup (사전 준비)

Phase 1에서는 현장 방문 전에 서비스 구역 정의, 경로 작성, 경로 검토를 수행합니다.

<Image src={workspacePhase1Empty} alt="Workspace Phase 1 Job 미선택 — 좌측에 Job Queue와 New Job 버튼, 우측에 Select a Job 안내 메시지" />

- **좌측 — Job Queue**: 생성된 Job 목록이 표시됩니다. 상단의 **New Job** 버튼을 클릭하여 새 Job을 생성할 수 있습니다.
- **우측 — Step 영역**: Job을 선택하지 않은 상태에서는 "Select a Job" 안내 메시지가 표시됩니다.

<Image src={workspacePhase1Selected} alt="Workspace Phase 1 Job 선택 시 — 3개 Step 카드: Add/Expand Service Area, Create Topological Path, Review & Apply Path" />

Job을 선택하면 Phase 1의 3개 Step이 카드 형태로 표시됩니다:

1. **Add/Expand Service Area**: 서비스 구역을 등록하거나 확장합니다. Doc, Action, Complete 상태가 단계별로 표시됩니다.
2. **Create Topological Path**: 로봇 이동 경로를 생성합니다. **Open Tool** 버튼을 클릭하여 Path Editor를 실행합니다.
3. **Review & Apply Path**: 작성된 경로를 검토하고 적용합니다.

:::note
각 Step 카드에는 진행 상태 아이콘이 표시됩니다. 초록색 체크 아이콘은 완료를 의미하며, 진행 중인 Step은 별도의 상태 표시가 나타납니다. **Open Tool** 버튼을 클릭하면 해당 Step의 에디터 도구로 바로 이동할 수 있습니다.
:::

## Phase 2 — On-site Installation (현장 설치)

Phase 2에서는 현장 매핑 데이터를 수집하고, 데이터 처리 및 맵 생성 파이프라인을 실행합니다.

<Image src={workspacePhase2} alt="Workspace Phase 2 — Introduction, Data Processing, Map Processing, Map Building 4개 그룹으로 Step 카드가 분류됨" />

Phase 2의 Step들은 4개 중분류로 그룹화되어 표시됩니다:

- **Introduction**: 매핑 데이터 수집 및 기초 설정을 담당합니다.
  - Base Map Logging, Upload & Download, Base Map Setting
- **Data Processing**: 수집된 데이터를 처리하는 서버 자동화 단계입니다.
  - Data Preprocessing, Segmentation Inference, 3D Map Aggregation
- **Map Processing**: 생성된 맵을 보정하고 2D 맵을 생성합니다.
  - 3D Map Correction, 2D Map Generation
- **Map Building**: 최종 맵을 구성하고 검수합니다.

각 Step 카드에는 현재 상태(진행 중, 완료 등)가 표시되며, **Open Tool** 버튼으로 해당 도구를 바로 실행할 수 있습니다.

:::caution
Data Processing 그룹의 Step들은 서버에서 자동으로 처리됩니다. 처리 중에는 해당 Step 카드가 In-Progress 상태로 표시되며, 완료되면 자동으로 다음 단계로 전환됩니다.
:::

## Phase 3 — Area Expansion (지역 확장)

Phase 3에서는 기존 맵에 새로운 구역을 추가합니다. 이 단계는 **세션(Session)** 단위로 관리되며, 여러 세션을 추가하여 맵을 반복적으로 확장할 수 있습니다.

<Image src={workspacePhase3} alt="Workspace Phase 3 — Active 섹션에 진행 중인 세션 카드, Done 섹션에 완료된 세션 카드가 표시됨" />

### 세션 관리

Phase 3 화면은 두 개의 섹션으로 나뉩니다:

- **Active**: 현재 진행 중인 세션이 표시됩니다. 각 세션 카드에는 포함된 Step 목록과 진행 상태가 나타납니다.
- **Done**: 완료된 세션이 표시됩니다.

각 세션 카드에서는 다음 작업을 수행할 수 있습니다:

- **Edit**: 세션 내용을 수정합니다.
- **Retry**: 실패한 Step을 재시도합니다.

:::tip
한 Job에 여러 세션을 추가하여 맵을 점진적으로 확장할 수 있습니다. 예를 들어, 세션 1에서 서쪽 구역을 추가하고, 세션 2에서 동쪽 구역을 추가하는 식으로 운영합니다. 각 세션의 타임스탬프로 생성 시각을 확인할 수 있습니다.
:::
