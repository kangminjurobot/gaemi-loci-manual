---

대시보드는 HD 맵 생성 워크플로를 관리하는 중앙 허브입니다. 모든 Job의 생성, 진행 상태 추적, 도구 실행을 한 곳에서 수행할 수 있습니다.

## 핵심 개념

| 개념 | 설명 |
|------|------|
| **Phase** | 워크플로의 대분류 단계. 총 3개 Phase로 구성됩니다. |
| **Job** | 하나의 서비스 구역에 대한 HD 맵 생성 작업 단위. 각 Job은 독립적인 워크플로를 가집니다. |
| **Step** | 워크플로 내 개별 작업 단계. 각 Phase에 여러 Step이 포함됩니다. |
| **Session** | Phase 3에서 맵을 반복 확장할 때의 개별 작업 단위입니다. |

세 개의 Phase는 다음과 같이 구성됩니다:

- **Phase 1 — Pre-installation (사전 준비)**: 서비스 구역을 정의하고, 로봇 경로를 작성하고 검토합니다.
- **Phase 2 — On-site Installation (현장 설치)**: 현장에서 매핑 데이터를 수집하고, 데이터 처리 및 맵 생성, 맵 편집을 수행합니다.
- **Phase 3 — Area Expansion (지역 확장)**: 기존 맵에 새로운 구역을 추가합니다. 세션 단위로 반복하여 맵을 점진적으로 확장합니다.
