충전 스테이션 설치는 크게 두 단계로 구성됩니다. 먼저 충전 스테이션 본체에 **고유 ID를 설정**한 뒤, 매핑 데이터(PCD)를 기반으로 **3D 공간상의 위치와 방향을 등록**합니다. 이 정보는 로봇이 자동으로 충전 스테이션에 도킹할 때 사용됩니다.

## 충전 스테이션 ID 셋팅

로봇을 충전 스테이션에 물리적으로 연결(충전)한 상태에서 ID를 설정합니다. ID는 각 충전 스테이션을 고유하게 식별하기 위한 값으로, 시스템 전체에서 중복되지 않아야 합니다.

### 1. 충전 스테이션 ID 획득

<Image src={screenshotIdAcquire} alt="충전 스테이션 본체에 부착된 라벨에서 물리적 ID를 확인하는 모습" />

충전 스테이션에 현재 설정되어 있는 ID를 조회합니다. 초기 설정값이거나 이전에 소프트웨어에서 설정한 ID가 표시됩니다.

### 2. 현재 ID 확인

<Image src={screenshotIdCheck} alt="Get ID 버튼을 클릭하여 현재 충전 스테이션에 설정된 ID를 확인하는 화면" />

**Get ID** 버튼을 클릭하면 현재 충전 스테이션에 설정되어 있는 ID를 조회할 수 있습니다. 새로 설치한 충전 스테이션이라면 기본값이 표시되거나 미설정 상태일 수 있습니다.

### 3. ID 셋팅

<Image src={screenshotIdSet} alt="ID 입력 필드에 원하는 충전 스테이션 ID를 기입하고 Set ID 버튼을 클릭하는 화면" />

입력 필드에 원하는 ID를 기입한 뒤 **Set ID** 버튼을 클릭합니다.

:::caution
ID를 변경하면 해당 충전 스테이션을 참조하는 기존 설정에 영향을 줄 수 있습니다. 운영 중인 시스템에서는 신중하게 변경하세요.
:::

### 4. 셋팅 성공

<Image src={screenshotIdSuccess} alt="충전 스테이션 ID 설정이 성공적으로 완료되었음을 나타내는 성공 메시지 화면" />

ID 설정이 정상적으로 완료되면 성공 메시지가 표시됩니다. **Get ID** 버튼으로 다시 한번 설정된 ID를 확인하는 것을 권장합니다.

## 충전 스테이션 위치 정보 셋팅

매핑 데이터(PCD)를 활용하여 충전 스테이션의 3D 위치와 방향을 등록합니다. 이 정보를 기반으로 로봇이 자동 도킹을 수행하므로, 정확한 위치 지정이 매우 중요합니다.

### 1. 데이터셋 로딩

<Image src={screenshotOpen} alt="툴바의 Open 버튼을 클릭하여 데이터셋 열기를 시작하는 화면" />

툴바의 **Open** 버튼을 클릭하여 매핑 데이터를 불러옵니다.

### 2. 팝업 창

<Image src={screenshotPopup} alt="데이터셋 선택을 위한 팝업 창이 표시된 화면" />

데이터셋을 선택할 수 있는 팝업 창이 표시됩니다.

### 3. Service Area 및 Local Time 선택

<Image src={screenshotServiceArea} alt="Service Area와 Local Time을 선택한 뒤 Load Dataset 버튼을 클릭하는 화면" />

충전 스테이션이 설치된 **Service Area**와 매핑이 수행된 **Local Time**을 선택한 뒤 **Load Dataset** 버튼을 클릭합니다.

:::tip
충전 스테이션이 명확하게 촬영된 시간대의 데이터를 선택하면 이후 위치 지정 작업이 수월합니다.
:::

### 4. 데이터셋 로드 완료

<Image src={screenshotDatasetLoaded} alt="데이터셋이 로드되어 키프레임 목록과 포인트클라우드가 표시된 화면" />

데이터셋이 로드되면 좌측 패널에 키프레임 목록이 표시됩니다. **화살표 키(상/하)**로 키프레임을 순차적으로 이동하며 포인트클라우드를 확인할 수 있습니다.

### 5. 전체 PCD 로딩

<Image src={screenshotLoadAllPcd} alt="Load All PCD 버튼을 클릭하여 전체 포인트클라우드를 로드하는 화면" />

개별 키프레임만으로 충전 스테이션을 찾기 어려울 경우, **Load All PCD** 버튼을 클릭하여 전체 포인트클라우드를 한꺼번에 로드합니다.

:::note
전체 PCD 로딩은 데이터셋 크기에 따라 시간이 소요될 수 있습니다. 로드가 완료될 때까지 기다려 주세요.
:::

### 6. 로딩 중

<Image src={screenshotLoading} alt="전체 포인트클라우드 로딩이 진행 중인 화면 — 1~2분 소요 가능" />

전체 PCD를 로딩하는 중입니다. 데이터 크기에 따라 **1~2분**이 소요될 수 있습니다. 로딩이 완료되면 3D 뷰포트에 전체 포인트클라우드가 표시됩니다.

### 7. 충전 스테이션 찾기

<Image src={screenshotFindStation} alt="전체 포인트클라우드에서 충전 스테이션 위치를 탐색하는 화면 — 마우스 드래그로 회전, 스크롤로 줌" />

전체 PCD가 로드된 상태에서 충전 스테이션의 위치를 찾습니다.

| 조작 | 동작 |
|------|------|
| **좌클릭 드래그** | 시점 회전 |
| **마우스 스크롤** | 확대/축소 |
| **우클릭 드래그** | 시점 이동 |

:::tip
충전 스테이션의 대략적인 설치 위치를 미리 파악해 두면 탐색 시간을 줄일 수 있습니다. 벽면 근처에 설치된 경우가 많으므로, 벽체 형상을 기준으로 탐색하면 효과적입니다.
:::

### 8. Charge+ 버튼 클릭

<Image src={screenshotAddButton} alt="Charge+ 버튼을 클릭하여 충전 스테이션 추가 모드로 전환하는 화면" />

툴바의 **+Charge** 버튼을 클릭하면 커서가 **'+'** 형태로 변경됩니다. 이 상태에서 3D 뷰포트의 충전 스테이션 위치를 정확히 클릭합니다.

### 9. 3D 모델 확인

<Image src={screenshot3dModel} alt="충전 스테이션 위치 클릭 후 3D 모델이 해당 위치에 표시된 화면" />

클릭한 위치에 충전 스테이션의 **3D 모델**이 표시됩니다. 모델의 위치와 방향이 실제 설치 상태와 일치하는지 확인합니다.

### 10. 미세 조정

<img src={screenshotFineAdjust.src} alt="충전 스테이션 3D 모델의 위치와 방향을 미세 조정하는 과정 — Rotation으로 방향, Translation으로 위치 이동" />

충전 스테이션 모델의 위치와 방향을 정밀하게 조정합니다.

| 조정 항목 | 설명 |
|-----------|------|
| **Rotation** | 충전 스테이션의 방향(각도)을 조정합니다 |
| **Translation** | 충전 스테이션의 위치(좌표)를 이동합니다 |

:::caution
충전 스테이션의 방향이 실제 설치 방향과 정확히 일치해야 합니다. 방향이 어긋나면 로봇의 **자동 도킹이 정상적으로 동작하지 않을 수 있습니다**. 포인트클라우드와 3D 모델을 꼼꼼히 비교하며 조정하세요.
:::

### 11. Edge 설정

<Image src={screenshotEdgeSetting} alt="충전 스테이션이 설치된 Edge(경로 구간)를 선택하는 화면" />

충전 스테이션이 설치된 **Edge(경로 구간)**를 선택합니다. Edge는 로봇이 충전 스테이션에 접근할 때 사용하는 경로 구간을 의미합니다. 올바른 Edge를 지정해야 로봇이 정확한 경로로 도킹에 진입할 수 있습니다.

### 12. Add 확정

<Image src={screenshotAddConfirm} alt="충전 스테이션 정보를 최종 확인한 뒤 Add 버튼을 클릭하여 등록을 확정하는 화면" />

설정한 위치, 방향, Edge 정보를 확인한 뒤 **Add** 버튼을 클릭하여 충전 스테이션 등록을 확정합니다.

### 13. Complete Step

<Image src={screenshotComplete} alt="모든 충전 스테이션 등록을 완료하고 Complete Step 버튼을 클릭하는 화면" />

현장에 설치된 모든 충전 스테이션의 등록이 완료되면 **Complete Step** 버튼을 클릭하여 다음 단계로 진행합니다.
